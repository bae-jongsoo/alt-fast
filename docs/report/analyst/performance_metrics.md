# 핵심 트레이딩 성과 지표

## 개요

트레이딩 전략의 품질을 정량적으로 평가하기 위한 핵심 지표들을 정리한다.
각 지표의 공식, 해석 방법, 한계점, 그리고 **실전 의사결정에서의 활용법**을 포함한다.

---

## 1. 승률 (Win Rate)

### 공식
```
Win Rate = 이익 거래 수 / 전체 거래 수 × 100
```

### 해석
- 전체 매매 중 수익을 낸 비율
- 단독으로는 전략의 수익성을 판단할 수 없음 (손익비와 반드시 함께 봐야 함)

### 벤치마크
| 구간 | 해석 |
|------|------|
| 60% 이상 | 높은 승률, 단 손익비가 낮을 가능성 |
| 40~60% | 일반적 범위, 손익비에 따라 수익/손실 결정 |
| 40% 미만 | 낮은 승률, 높은 손익비가 필수 |

### 한계점
- **승률 80%여도 1번의 큰 손실이 8번의 소익을 다 지울 수 있음**
- 승률만 높이려면 손절을 안 하면 되는데, 이는 치명적 리스크
- 소수 샘플에서 승률은 통계적 의미가 약함 (최소 30~50거래 필요, 이상적으로 100거래 이상)

### 의사결정 연결
- 승률이 꾸준히 50% 미만이면 → 진입 조건(매수 프롬프트) 점검
- 승률이 높은데 수익이 안 나면 → 손익비 문제, 매도 타이밍 점검

---

## 2. 평균 이익/손실 (Average Win / Average Loss)

### 공식
```
평균 이익 = Σ(이익 거래의 수익) / 이익 거래 수
평균 손실 = Σ(손실 거래의 손실) / 손실 거래 수
손익비(Risk-Reward Ratio) = 평균 이익 / |평균 손실|
```

### 해석
- 이길 때 얼마나 크게 이기고, 질 때 얼마나 적게 지는지
- 손익비 > 1이면 이익이 손실보다 크다는 뜻

### 벤치마크
| 손익비 | 최소 필요 승률 (손익분기) |
|--------|--------------------------|
| 1:1 | 50% |
| 1:2 | 33% |
| 1:3 | 25% |
| 2:1 | 67% |

### 한계점
- 평균이므로 극단적인 한 건의 대박/폭망에 왜곡될 수 있음
- 중앙값(median)과 함께 보면 분포를 더 정확히 파악 가능

### 의사결정 연결
- 손익비가 낮으면 → 매도 전략 점검 (너무 빨리 팔고 있는지)
- 평균 손실이 크면 → 손절 기준 점검

---

## 3. 기대값 (Expectancy)

### 공식
```
기대값 = (승률 × 평균 이익) - (패률 × |평균 손실|)
```

### 예시
```
승률 55%, 평균 이익 450원, 평균 손실 300원일 때:
기대값 = (0.55 × 450) - (0.45 × 300) = 247.5 - 135 = +112.5원/거래
```

### 해석
- **양수면 장기적으로 수익, 음수면 장기적으로 손실**
- 한 거래당 평균적으로 기대할 수 있는 수익/손실
- 승률과 손익비를 하나의 숫자로 통합한 핵심 지표

### 한계점
- 과거 데이터 기반이므로 미래를 보장하지 않음
- 시장 레짐(상승장/하락장/횡보장) 변화에 따라 급변 가능
- 충분한 샘플 수가 필요 (최소 50~100거래)

### 의사결정 연결
- **기대값이 보고서의 가장 중요한 단일 숫자**
- 기대값이 음수면 전략 전체를 재검토해야 함
- 일별 기대값 추이를 보면 전략 품질의 변화를 추적 가능

---

## 4. Profit Factor (총이익/총손실)

### 공식
```
Profit Factor = 총 이익(Gross Profit) / |총 손실(Gross Loss)|
```

### 대안 공식
```
Profit Factor = (승률 × 평균 이익) / (패률 × |평균 손실|)
```

### 벤치마크
| 구간 | 해석 |
|------|------|
| < 1.0 | 손실 전략 (총 손실 > 총 이익) |
| 1.0~1.2 | 마진 전략, 수수료/슬리피지로 손실 전환 위험 |
| 1.2~1.5 | 약한 양의 에지 |
| 1.5~2.5 | 전문 트레이더 목표 범위 |
| 2.5~3.0 | 우수 |
| > 3.0 | 예외적, **소수 샘플 또는 과적합 의심** |

### 한계점
- 단일 대형 이익 거래가 결과를 왜곡 가능
- 거래 수가 적으면 의미 없음
- Profit Factor > 3.0은 오히려 의심 (소수 샘플, 생존 편향 가능성)

### 의사결정 연결
- PF가 1.0~1.2 범위면 세후/수수료 고려 시 실질 손실일 수 있음
- **report_plan.md의 5번 항목(승률/손익비)에서 PF를 핵심 지표로 사용**

## 5. Sharpe Ratio (위험 조정 수익률)

### 공식
```
Sharpe Ratio = (Rp - Rf) / σp

Rp = 포트폴리오 수익률 (일별 수익률의 평균)
Rf = 무위험 수익률 (한국: 91일 CD금리 또는 국고채 3년물)
σp = 포트폴리오 수익률의 표준편차
```

### 일별 → 연간 환산
```
연간 Sharpe = 일별 Sharpe × √252
(252 = 연간 거래일 수, 한국은 약 247일이나 252를 관행적으로 사용)
```

### 벤치마크 (연간 기준)
| 구간 | 해석 |
|------|------|
| < 0 | 무위험 수익률보다 낮음 |
| 0~0.5 | 낮음 |
| 0.5~1.0 | 보통 |
| 1.0~2.0 | 좋음 (전문 전략 최소 기준) |
| 2.0~3.0 | 우수 |
| > 3.0 | 예외적 (과적합 의심) |

### 한계점
- **상방 변동성과 하방 변동성을 동일하게 취급** → 큰 이익도 페널티
- 수익률의 정규분포를 가정하지만, 실제 트레이딩 수익은 두꺼운 꼬리(fat tail)
- 일중 매매(데이 트레이딩)에서는 일별 수익률 계산 자체가 단순해져서 의미가 줄어듦
- 소수 거래일에서는 표준편차 추정이 부정확

### 의사결정 연결
- 주간/월간 단위로 추적하여 전략 품질의 시계열 변화 모니터링
- 일일 보고서에서는 참고 지표로, 주간/월간 보고서에서 핵심 지표로 사용

---

## 6. Sortino Ratio (하방 위험 조정 수익률)

### 공식
```
Sortino Ratio = (Rp - Rf) / σd

σd = 하방 편차(Downside Deviation) = √(Σ min(Ri - Rf, 0)² / n)
```

### Sharpe vs Sortino
| 비교 항목 | Sharpe | Sortino |
|-----------|--------|---------|
| 변동성 기준 | 전체 (상방+하방) | 하방만 |
| 큰 이익에 대해 | 페널티 부여 | 페널티 없음 |
| 실무 적합성 | 전통적 사용 | **트레이딩에 더 적합** |

### 해석
- Sortino가 Sharpe보다 높으면: 수익률 변동의 대부분이 상방(이익)에서 발생
- Sortino가 Sharpe와 비슷하면: 상방/하방 변동이 비슷
- **기관 트레이더들은 Sortino를 Sharpe보다 더 유용하다고 평가**

### 한계점
- Sharpe와 동일한 소수 샘플 문제
- 하방 편차 계산 시 손실 거래가 매우 적으면 불안정

### 의사결정 연결
- Sharpe < Sortino이면 → 전략이 상방 변동성이 큰 것 (좋은 신호)
- Sharpe > Sortino이면 → 하방 리스크가 상방보다 큼 (위험 신호)

## 7. Maximum Drawdown (MDD, 최대 낙폭)

### 공식
```
Drawdown(t) = (Peak(t) - Equity(t)) / Peak(t) × 100
MDD = max(Drawdown(t)) for all t
```

### 일중(Intraday) MDD
```
일중 MDD = 당일 내 평가자산 최고점 대비 최대 하락폭
```

### 계산 방법
1. 시간순으로 자산 가치를 추적
2. 각 시점에서의 고점(high-water mark) 갱신
3. 현재 가치와 고점의 차이가 낙폭(drawdown)
4. 전체 기간 중 가장 큰 낙폭이 MDD

### 두 가지 MDD 관점
| 유형 | 설명 | 용도 |
|------|------|------|
| 일중(Intraday) MDD | 미실현 손익 포함, 장중 순간 최대 낙폭 | 리스크 스트레스 측정 |
| 종가(Close-to-Close) MDD | 실현된 거래 기준 | 실질적 자산 변화 측정 |

### 한계점
- 단일 숫자로 전체 리스크를 대표하기 어려움
- 과거 MDD가 미래 최악을 보장하지 않음
- 회복 시간(Recovery Time)과 함께 봐야 완전한 그림

### 의사결정 연결
- **report_plan.md 10번 항목(자산 변동 곡선)의 핵심 파생 지표**
- 일중 MDD가 사전 설정 한도를 초과하면 → 리스크 관리 강화 필요
- MDD 대비 수익률(Calmar Ratio = 연간수익/MDD)로 위험 대비 보상 평가

## 8. 벤치마크 대비 성과 (Alpha / Beta)

### Beta (시장 민감도)
```
Beta = Cov(Rp, Rm) / Var(Rm)

Rp = 전략 일별 수익률
Rm = 벤치마크(코스피/코스닥) 일별 수익률
```

### Alpha (초과 수익)
```
Alpha = Rp - (Rf + Beta × (Rm - Rf))

간이 계산 (일별):
Alpha(일별) = 전략 수익률 - 벤치마크 수익률
```

### 해석
| Alpha | 의미 |
|-------|------|
| > 0 | 시장 대비 초과 수익 (실력) |
| = 0 | 시장과 동일 성과 |
| < 0 | 시장 대비 부진 (시장 ETF 사는 게 나았음) |

| Beta | 의미 |
|------|------|
| > 1 | 시장보다 변동 큼 (공격적) |
| = 1 | 시장과 동일 변동 |
| < 1 | 시장보다 변동 작음 (방어적) |
| < 0 | 시장과 반대 방향 (매우 드묾) |

### 한계점
- 일별 Alpha는 변동이 크므로 단기적으로 의미 제한적
- 벤치마크 선택이 중요 (코스피 vs 코스닥 vs 섹터 지수)
- 단기 데이 트레이딩에서는 Beta가 구조적으로 낮을 수 있음 (시장 노출 시간이 짧으므로)

### 의사결정 연결
- **report_plan.md 8번 항목(코스피/코스닥 대비 수익률)의 핵심 프레임워크**
- Alpha가 지속적으로 양수인지 추적 → 전략의 존재 이유 증명
- 상승장에서만 Alpha 양수, 하락장에서 음수면 → 방향성 편향 있음

## 9. 변동성 캡처율 (Volatility Capture Ratio)

### 공식
```
캡처율 = 실제 수익 / 종목 일중 변동폭 × 100

종목 일중 변동폭 = (당일 고가 - 당일 저가) / 당일 저가 × 100
```

### 해석
- 시장이 제공한 기회 대비 실제로 캡처한 수익의 비율
- 절대 수익이 아니라 **기회 대비 효율성** 측정

### 의사결정 연결
- **report_plan.md 6번 항목(변동성 대비 성과)의 핵심 지표**
- 변동성이 큰 날 캡처율이 낮으면 → 진입/청산 타이밍 문제
- 종목별 캡처율 비교로 어떤 종목에서 전략이 잘 작동하는지 파악

## 10. Kelly Criterion (최적 포지션 사이징)

### 공식
```
Kelly % = (W × R - L) / R

W = 승률 (소수)
L = 패률 (1 - W)
R = 손익비 (평균 이익 / |평균 손실|)
```

### 예시
```
승률 55%, 손익비 1.5일 때:
Kelly % = (0.55 × 1.5 - 0.45) / 1.5 = (0.825 - 0.45) / 1.5 = 25%
```

### 실전 적용
- **Full Kelly는 너무 공격적** → 실무에서는 Half Kelly(50%) 또는 Quarter Kelly(25%) 사용
- 대부분의 프로 트레이더는 거래당 2~4% 리스크

### 한계점
- 승률/손익비 추정이 조금만 빗나가도 포지션이 과대해짐
- 연속 손실 시 급격한 자산 감소 위험
- 독립적 거래를 가정하지만, 실제로는 종목 간 상관관계 존재

### 의사결정 연결
- 현재 포지션 사이징이 Kelly 대비 어디인지 확인
- Kelly를 초과하는 포지션은 리스크 과다

## 지표 간 관계 요약

```
기대값 = f(승률, 손익비)
Profit Factor = 기대값의 비율 버전
Sharpe/Sortino = 수익률의 변동성 대비 평가
MDD = 최악의 시나리오 리스크
Alpha = 시장 대비 초과 성과
Kelly = 위 지표들에 기반한 최적 베팅 크기
```

### 보고서에서의 우선순위

| 순위 | 지표 | 이유 |
|------|------|------|
| 1 | 기대값(Expectancy) | 전략의 수익성을 단일 숫자로 요약 |
| 2 | Profit Factor | 이익/손실 비율의 직관적 표현 |
| 3 | 일중 MDD | 당일 리스크 수준 즉시 파악 |
| 4 | 승률 + 손익비 | 기대값의 구성 요소 분해 |
| 5 | Alpha (벤치마크 대비) | 시장 대비 실력 측정 |
| 6 | Sharpe/Sortino | 주간/월간 추적용 |
| 7 | Kelly % | 포지션 사이징 적정성 점검 |
