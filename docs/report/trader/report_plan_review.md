# 보고서 계획 트레이더 관점 리뷰

> `docs/report_plan.md`의 11개 분석 항목에 대해, 국내 단타매매 전문가 관점에서 보완/추가할 사항을 정리한다.

---

## 1. 항목별 보완 사항

### 항목 1: 종목별 매매 타임라인
**현재**: 몇시에 사서 몇시에 팔았는지, 보유시간, 손익

**보완**:
- **보유시간 구간 분류**: ~10분(초단타), ~30분(단타), ~1시간(스윙단타), 1시간+(중기)별 성과 집계
- **장 시간대 태그**: 각 매매에 "장초반/오전장/점심/오후장/마감전" 태그 부여
- **진입 속도**: 판단(decision) → 체결(executed) 사이 지연시간 표시

### 항목 2: 놓친 기회 분석
**현재**: 보유 중 최고가 vs 실제 매도가 (고점 캡처율)

**보완**:
- **일중 MDD**: 보유 구간 내 최저가 vs 매수가. "매수 후 얼마나 빠졌다가 올라왔는지"
  - 고점 캡처율이 높아도, 중간에 -3%까지 빠졌다면 심리적 리스크가 높은 매매
- **저점 회피율**: (매수가 - 당일저가) / (당일고가 - 당일저가) → "얼마나 좋은 가격에 샀는지"
- **시간 기반 최적점**: "10분만 더 기다렸으면", "5분 일찍 팔았으면" 분석

### 항목 3: 시간대별 수익 분석
**현재**: 1시간 단위로 그룹핑

**보완**:
- **6구간 세분화** (트레이더 시간대):
  1. 09:00~09:30 (장초반/골든타임)
  2. 09:30~11:30 (오전장)
  3. 11:30~13:00 (점심시간대)
  4. 13:00~14:30 (오후장)
  5. 14:30~15:20 (마감 접근)
  6. 15:20~15:30 (종가 동시호가)
- **시간대별 경고**: 점심시간대 매매가 3건 이상이면 경고 (비효율 구간)
- **시간대별 추천**: "오전장 승률이 가장 높습니다. 이 시간대 매매 집중을 권장합니다."

### 항목 4: HOLD 판단 복기
**현재**: HOLD 판단 후 30분/60분 후 가격 변동

**보완**:
- **HOLD 판단 시점의 호가 상태**: HOLD 판단 당시 수급이 매수 우위였는지 매도 우위였는지
- **HOLD vs 선택된 종목 비교**: "A종목 HOLD → 30분 후 +2%, B종목 BUY → 결과 -0.5%"
  - 종목 선택 자체의 품질 측정
- **HOLD 빈도**: HOLD가 너무 많으면(>70%) 보수적, 너무 적으면(<20%) 공격적

### 항목 5: 승률 / 손익비
**현재**: 승률, 평균이익/손실, Profit Factor, 최대 단일 이익/손실

**보완**:
- **기대값 (Expectancy)**: (승률 × 평균이익) - (패률 × 평균손실) → 건당 기대 수익
- **Kelly Criterion**: 최적 투입 비율 계산 → 현재 투입 비율과 비교
- **실질 손익비**: 수수료+세금 반영 후 실질 손익비 (표면 vs 실질 괴리)
- **연속 승/패**: 최대 연속 승, 최대 연속 패 → 멘탈 리스크 측정
- **시간대별 승률**: 장초반/오전/오후별 승률 차이

### 항목 6: 변동성 대비 성과
**현재**: 당일 고가-저가 범위 대비 실현 수익 비율

**보완**:
- **ATR 기반 캡처율**: 분봉 기반 ATR 계산 → 더 정밀한 변동성 측정
  ```
  캡처율 = |실현수익| / ATR(14)
  ```
- **변동성-수익 산점도**: X축=당일 변동폭, Y축=실현수익 → 상관계수 계산
- **변동성 구간별 성과**: 저변동(<1%), 중변동(1~3%), 고변동(>3%) 구간별 승률
- **효율비**: (실현수익 / 보유시간) → 시간당 수익 효율

### 항목 7: LLM 판단근거 복기
**현재**: 소스별 가중치와 해당 소스 의존 판단의 승패율

**보완 없음** — 현재 설계로 충분. 다만 분석 시 호가 소스의 가중치 추적을 별도로 강조할 것.

### 항목 8: 코스피/코스닥 대비 수익률
**현재**: 시장 등락률 vs 시스템 수익률 비교

**보완**:
- **종목별 알파**: 개별 종목의 당일 등락률 대비 해당 종목에서 얻은 수익률
  - 종목이 +3% 올랐는데 +1%만 먹었으면, 종목 알파는 -2%
- **시장 상태 분류**: 상승장(>1%), 횡보(-1%~1%), 하락장(<-1%)별 시스템 성과
  - "하락장에서도 수익을 내는가?" → 진짜 알파 측정

### 항목 9: 동일종목 반복매매 패턴
**현재**: 회차별 수익률 변화

**보완**:
- **경고 기준 수치화**: 3회 이상 동일종목 매매 시 경고 플래그
- **누적 수수료 표시**: 반복매매의 수수료 총합 vs 수익 총합
- **최적 회차**: "이 종목은 2회까지가 최적, 3회부터 손실 전환" 패턴 분석
- **LLM 편향 감지**: 특정 종목 선택 빈도가 비정상적으로 높으면 프롬프트 편향 의심

### 항목 10: 자산 변동 곡선
**현재**: 시계열 평가자산 변화, 일중 최대 낙폭

**보완**:
- **회복 시간**: MDD 발생 후 원금 복귀까지 소요 시간
- **연속 하락 구간**: 자산이 연속으로 줄어든 최장 구간 (시간)
- **리스크 조정 수익**: Sharpe Ratio (일중 버전) = 수익률 / 수익률 표준편차
- **일일 한도 경고**: MDD가 설정 한도(예: -3%) 초과 시 빨간색 경고

### 항목 11: 호가 수급 분석
**현재**: 매매 시점의 호가창 상태와 성과의 관계

**보완**:
- **체결강도 연동**: 호가 잔량뿐 아니라 체결강도(매수체결/매도체결)도 함께 분석
- **허매수/허매도 감지**: 연속 스냅샷 비교로 급변 잔량 탐지
- **호가 두께 구간별 성과**: 얇은 호가 vs 두꺼운 호가 시점 매매의 슬리피지 차이
- **1호가 집중도**: 1호가에 잔량이 몰려있는 정도와 체결 속도의 관계

## 2. 추가 항목 제안

### 항목 12: 매매 빈도 적정성 분석 (신규 제안)

**뭘 보나**: 하루 총 매매 횟수, 판단 횟수 대비 실행 비율, 시간당 매매 밀도

**왜 필요한가**:
- 너무 잦은 매매 → 수수료 누적, 의사결정 품질 저하
- 너무 적은 매매 → 기회 놓침, 자금 효율 저하
- **적정 매매 빈도**를 찾는 것이 자동매매 시스템 최적화의 핵심

**분석 예시**:
```
[매매 빈도 분석]
- 총 판단 횟수: 78회 (BUY 판단 시도)
- 실제 매수 실행: 12회 (실행률 15.4%)
- HOLD 판단: 66회 (84.6%)
- 시간당 평균 매매: 1.8건
- 수수료 총액: 41,400원 (순이익의 32.1%)

→ 수수료 비중이 30%를 초과합니다.
  매매 빈도를 줄이거나 목표 수익폭을 키우는 것을 권장합니다.
```

### 항목 13: 매수가 최적성 분석 (신규 제안)

**뭘 보나**: 실제 매수가 vs 당일 최저가의 관계

**왜 필요한가**:
- "좋은 종목을 골랐지만, 비싸게 산" 경우 식별
- 진입 타이밍의 품질 측정
- 매도뿐 아니라 매수 타이밍도 평가해야 전략 전체를 최적화 가능

**분석 예시**:
```
[매수 타이밍 분석]
- 평균 매수 위치: 상위 62% (0%=최저가, 100%=최고가)
- 최적 매수 비율: 3/12건 (25%) — 하위 30% 가격대에서 진입 성공
- 평균 매수 후 추가 하락: -0.4% (진입 후 더 빠진 평균 폭)

→ 매수 타이밍이 약간 높은 편입니다.
  진입 조건에 "직전 1~2분봉 하락 확인 후 반등" 조건 추가를 검토하세요.
```
