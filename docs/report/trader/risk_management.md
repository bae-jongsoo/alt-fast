# 리스크 관리 기법 — 단타매매 관점

## 1. 손절 (Stop Loss) 전략

### 1.1 고정 비율 손절
- **기본 원칙**: 매수가 대비 -1%~-2% 도달 시 무조건 손절
- **현 시스템 수수료 고려**:
  - 매수 수수료: 0.015%
  - 매도 수수료: 0.015%
  - 거래세: 0.2% (매도 시)
  - **총 비용: ~0.23%**
  - 따라서 실질 손절선은 표면 -1%에서 실질 -1.23%

### 1.2 ATR 기반 동적 손절
```
손절가 = 매수가 - (ATR × 배수)
- 보수적: 1.5 × ATR
- 공격적: 1.0 × ATR
- 표준:   2.0 × ATR
```

- ATR(Average True Range): 일정 기간 동안의 평균 변동폭
- 변동성이 큰 종목은 손절 폭을 넓게, 작은 종목은 좁게 → 시장 적응형
- **보고서 적용**: 매매 시점의 분봉 기반 ATR을 계산하여, "ATR 대비 손실 비율"을 추적

### 1.3 시간 손절
- 진입 후 일정 시간(예: 30분, 1시간) 내 목표 수익 미달 시 청산
- **근거**: 단타에서 시간이 지날수록 예측 정확도가 감소
- **현 시스템 관점**: LLM 매도 판단에 "보유시간"을 더 강하게 반영할 필요

### 손절 기준 비교표

| 방식 | 장점 | 단점 | 현 시스템 적합도 |
|------|------|------|---------------|
| 고정 비율 (-1.5%) | 단순, 명확 | 변동성 미반영 | 중 |
| ATR 기반 | 시장 적응형 | 계산 복잡 | 상 (분봉 데이터 있음) |
| 시간 손절 (30분) | 자금 효율 | 반등 놓칠 수 있음 | 상 (보유시간 추적 가능) |
| 호가 기반 | 실시간 수급 반영 | 허매수 위험 | 중 (호가 스냅샷 있음) |

## 2. 포지션 사이징 (Position Sizing)

### 2.1 고정 금액법
- 1회 매매에 투입하는 금액을 고정
- 예: 총 자산 1,000만원 중 매매당 최대 300만원

### 2.2 고정 비율법 (Fixed Fractional)
```
투입금액 = 총자산 × 리스크비율 / 손절폭
```
- **리스크비율**: 1회 매매에서 감수할 수 있는 최대 손실 비율 (보통 1~2%)
- 예: 총자산 1,000만원, 리스크 2%, 손절폭 1.5%
  - 투입금액 = 1,000만 × 0.02 / 0.015 = 1,333만원 (총자산 초과 → 1,000만원으로 제한)

### 2.3 켈리 공식 (Kelly Criterion)
```
최적 베팅 비율 = (승률 × 평균이익 - 패률 × 평균손실) / 평균이익
             = 승률 - (1 - 승률) / 손익비
```
- 이론적 최적값이지만 실전에서는 **Half Kelly** (50%) 적용 권장
- **보고서 적용**: 누적 매매 데이터로 Kelly 값을 계산하여 적정 투입 비율 제안

### 현 시스템의 포지션 사이징
- 현재: LLM이 매수 수량을 직접 결정 (프롬프트에 cash_amount 제공)
- 한계: LLM의 수량 결정이 리스크 관리 원칙에 부합하는지 검증 필요
- **보고서에서 확인할 것**: 실제 투입 비율(매수금액/보유현금)의 분포와 그에 따른 성과

## 3. 최대 낙폭 관리 (MDD - Maximum Drawdown)

### MDD란?
```
MDD = (최고점 - 최저점) / 최고점 × 100%
```
- 투자 기간 중 최고 자산에서 최대로 떨어진 비율
- **일중 MDD (Intraday MDD)**: 하루 내 자산의 최대 낙폭

### 일일 최대 손실 한도
- **원칙**: 하루 총 손실이 일정 한도를 초과하면 당일 매매 중단
- 일반적 기준: 일일 자산의 -2%~-3%
- **현 시스템 관점**: 현재 일일 손실 한도 설정이 없음 → 보고서에서 일중 MDD를 추적하여 필요성 검증

### 보고서 항목 10번(자산 변동 곡선) 보완

```
[일중 리스크 지표]
- 일중 최대 낙폭 (Intraday MDD): -2.3%
  → 10:15 1,000만원 → 10:47 977만원 (최저점) → 15:30 1,012만원
- 일중 최대 수익: +3.1%
- 최종 수익: +1.2%
- 리스크/리워드: 1:1.35 (최종수익/최대낙폭)
- 회복 시간: 32분 (최저점 → 원금 복구)

[경고]
⚠ 일중 MDD가 2%를 초과했습니다.
  리스크 관리 기준 재검토를 권장합니다.
```

## 4. 승률과 손익비의 관계

### 기본 공식

```
기대값 (Expectancy) = (승률 × 평균이익) - (패률 × 평균손실)
Profit Factor = 총이익 / 총손실
손익비 = 평균이익 / 평균손실
최소 필요 승률 = 1 / (1 + 손익비)
```

### 손익비별 최소 필요 승률

| 손익비 | 최소 필요 승률 | 실전 의미 |
|--------|-------------|---------|
| 1:1 | 50% | 동전 던지기 이상이어야 수익 |
| 1.5:1 | 40% | 승률 40%면 충분히 수익 가능 |
| 2:1 | 33.3% | 3번 중 1번만 이겨도 수익 |
| 3:1 | 25% | 4번 중 1번 승리로 수익 |

### 현 시스템 수수료를 고려한 실질 손익비

```
실질 이익 = 매매이익 - (매수수수료 + 매도수수료 + 거래세)
실질 손실 = 매매손실 + (매수수수료 + 매도수수료 + 거래세)

예시 (75,000원 종목, 100주):
- 매수금액: 7,500,000원
- 총 비용: 7,500,000 × 0.0023 = 17,250원
- +1% 이익 시 실질: 75,000 - 17,250 = 57,750원 (+0.77%)
- -1% 손실 시 실질: -75,000 - 17,250 = -92,250원 (-1.23%)
- 실질 손익비: 0.77 / 1.23 = 0.63:1 (불리!)
```

**핵심 시사점**: 수수료+세금 때문에 표면 손익비 1:1이어도 실질적으로는 불리.
최소 표면 손익비 1.5:1 이상을 목표로 해야 실질적으로 수익 가능.

### 보고서 항목 5번(승률/손익비) 보완 제안

```
[승률/손익비 분석]
- 총 매매: 12건 (승 7, 패 4, 무 1)
- 승률: 58.3% (무 제외 63.6%)
- 평균 이익: +32,450원 (세후)
- 평균 손실: -28,100원 (세후)
- 손익비: 1.15:1
- Profit Factor: 1.55
- 기대값 (Expectancy): +7,410원/건

[리스크 평가]
- 최대 연속 승: 4건 (09:30~11:20)
- 최대 연속 패: 2건 (13:45~14:30)
- Kelly Criterion: 14.2% → Half Kelly 추천 투입비율: 7.1%
- 현재 평균 투입비율: 28.5% → ⚠ Kelly 기준 대비 과다 투입
```

## 5. 동일종목 반복매매(복수매매) 리스크

### 위험성
1. **감정적 집착**: 한 종목에 대한 과신 → 손실 후 "복수 매매"
2. **수수료 누적**: 반복 매매할수록 총 수수료 비중 증가
3. **회차별 성과 저하 패턴**: 통계적으로 동일종목 3회차 이후 수익률 급감하는 경향
4. **자동매매 특유 위험**: LLM이 같은 종목을 "익숙해서" 반복 선택할 가능성

### 보고서 항목 9번(반복매매) 보완 제안

```
[동일종목 반복매매 분석]

삼성전자(005930):
  1차: 09:15 매수 → 10:02 매도 | +0.8% (세후 +0.57%)
  2차: 10:45 매수 → 11:30 매도 | +0.3% (세후 +0.07%)
  3차: 13:20 매수 → 14:15 매도 | -0.5% (세후 -0.73%) ⚠

  → 회차별 수익 체감 패턴 감지
  → 3회차 이후 손실 전환
  → 해당 종목 총 수익: -0.09% (반복하지 않았으면 +0.57%)

[경고]
⚠ 동일종목 3회 이상 반복매매가 감지되었습니다.
  반복매매 시 수익이 감소하는 패턴이 확인됩니다.
  LLM 프롬프트에 "동일종목 반복 진입 자제" 가이드를 추가 검토하세요.
```
