# 스캘핑 전략 및 시간대별 매매 특성

## 1. 스캘핑/데이트레이딩 기본 개념

### 스캘핑 (Scalping)
- 초단타 매매. 수초~수분 내 매수/매도를 반복하여 소폭 차익을 누적
- 하루 수십~수백 회 거래. 1회당 수익은 작지만 횟수로 보상
- **핵심 역량**: 호가창 읽기, 체결 속도, 감정 통제

### 데이트레이딩 (Day Trading)
- 당일 매수/매도를 완결. 오버나잇 리스크 제거
- 1일 1~10회 수준 거래. 보유시간 수분~수시간
- **현재 시스템 방식**: 1종목 보유, 60초 주기 판단 → 사실상 데이트레이딩에 가까움

### 현 시스템과의 비교

| 항목 | 순수 스캘핑 | 현 시스템 (alt-fast) |
|------|-----------|---------------------|
| 판단 주기 | 수초 | 60초 (trading_interval) |
| 판단 주체 | 인간 직관 + 호가창 | LLM (프롬프트 기반) |
| 동시 보유 | 1~3종목 | 1종목 (get_open_position) |
| 진입 근거 | 호가/체결/틱차트 | 분봉 + 호가 + 뉴스 + 공시 + 재무 |
| 청산 근거 | 틱 움직임 | LLM 종합 판단 |

## 2. 시간대별 매매 특성 (한국 주식시장)

### 08:30~09:00 — 동시호가
- 전일 종가 이후 발생한 이벤트(뉴스, 해외 시장, 공시)가 반영되는 구간
- 매수/매도 주문을 모아서 09:00에 동시 체결 → **시가 결정**
- 갭 상승/갭 하락 종목 탐색 시간
- **보고서 관점**: 시가 갭 대비 당일 수익률을 추적하면, "갭 상승 종목에 진입한 건의 성과"를 분석 가능

### 09:00~09:30 — 장 초반 (골든타임)
- **거래량 최대 구간**. 하루 거래량의 20~30%가 이 시간에 집중
- 변동성이 가장 높음 → 기회와 위험이 공존
- 개인 투자자 중심의 감정적 매매가 많음 (전일 뉴스에 대한 반응)
- 기관/외국인은 이 구간에서 물량 확보(매집)하는 경우가 많음
- **보고서 관점**: 장초반 매매의 승률/손익비를 별도 집계해야 함. 시스템의 market_start_time이 09:11인 이유도 이 변동성 회피 가능성

### 09:30~11:30 — 오전장
- 변동성이 점차 줄어드는 구간
- 주요 추세가 형성되는 시간 (갭 후 방향 확정)
- 기관/외국인의 프로그램 매매가 본격화
- **보고서 관점**: 오전장 진입 건의 평균 보유시간과 수익률 추적

### 11:30~13:00 — 점심시간대
- 거래량 급감, 변동성 최저
- 호가 스프레드가 넓어짐 → 슬리피지 위험 증가
- 단타 매매에 비효율적인 구간
- **보고서 관점**: 이 시간대 매매가 있다면 성과를 별도 체크. 불필요한 매매 여부 검증

### 13:00~14:30 — 오후장
- 거래량이 다시 증가하기 시작
- 기관의 포지션 정리 매매가 시작됨
- 외국인의 동향이 명확해지는 시간
- **보고서 관점**: 오후장 전환 시점의 매매 패턴 분석

### 14:30~15:20 — 장 마감 접근
- 프로그램 매매 청산으로 인한 변동성 발생
- 당일 미체결 물량의 청산 매매
- 기관의 바스켓 매매(대량 매수/매도)
- **보고서 관점**: 마감 직전 매도 타이밍 평가. 시스템의 market_end_time=15:30이므로 이 구간 활용도 확인

### 15:20~15:30 — 종가 동시호가
- 종가 결정을 위한 동시호가
- 대형주 중심으로 기관/외국인 물량 집중

## 3. 스캘핑 진입/청산 전략

### 진입 조건 (BUY 시그널)

#### 호가창 기반
1. **매수잔량 >> 매도잔량** (수급 우위) + 체결강도 100% 이상
2. **매도벽 소화 후 돌파**: 큰 매도 잔량이 체결로 소화되면서 가격 상승
3. **허매도 판별**: 매도 잔량이 크지만 실제 체결은 매수 위주 → 세력의 물량 확보 신호

#### 분봉 기반 (현 시스템에 적합)
1. **양봉 연속 + 거래량 증가**: 1분봉에서 연속 양봉 + 직전 대비 거래량 2배 이상
2. **지지선 확인 후 반등**: 분봉 저점이 점차 높아지는 패턴
3. **이동평균선 골든크로스**: 단기(5분) MA가 중기(20분) MA를 상향 돌파

#### 뉴스/이벤트 기반
1. **긍정적 공시 + 거래량 폭증**: DART 공시 후 30분 이내 진입 검토
2. **섹터 모멘텀**: 관련 종목군 동반 상승 시

### 청산 조건 (SELL 시그널)

#### 익절 (Take Profit)
1. **목표가 도달**: 매수가 대비 +1.5%~3% (수수료+세금 고려 시 실질 +1%~2.5%)
2. **체결강도 급락**: 100% 이상 → 80% 이하로 전환
3. **매도잔량 급증**: 상위 호가에 대형 매도벽 출현

#### 손절 (Stop Loss)
1. **고정 손절**: 매수가 대비 -1%~-2% (절대 준수)
2. **ATR 기반 손절**: 1.5~2 × ATR 하회 시 청산
3. **시간 손절**: 진입 후 일정 시간(예: 30분) 내 목표 미달 시 청산 검토

## 4. 현 시스템에 대한 트레이더 관점 평가

### 강점
- **감정 배제**: LLM 판단으로 인간의 감정적 매매(FOMO, 공포) 제거
- **다중 소스 분석**: 분봉 + 호가 + 뉴스 + 공시 + 재무지표를 종합 → 인간 단타꾼보다 넓은 시야
- **일관성**: 동일 조건에서 동일 판단 (피로/멘탈 영향 없음)

### 약점 및 개선 포인트
1. **60초 주기의 한계**: 스캘핑에는 너무 느림. 다만 데이트레이딩에는 적절
2. **1종목 제한**: 기회 분산 불가. 좋은 종목을 놓칠 수 있음
3. **시간대별 전략 미분화**: 장초반/점심/장마감 동일 전략 → 시간대별 가중치 차별화 필요
4. **슬리피지 미반영**: 가상매매라 실제 체결 가격과 차이 발생 가능성

### 보고서에서 검증해야 할 것들
- 시간대별 승률 차이가 통계적으로 유의미한지
- 장초반(09:11~09:30) 매매 vs 나머지 시간대 매매의 성과 비교
- 보유시간이 길수록 수익이 나는지, 짧을수록 나는지
- 동일종목 반복 진입 시 회차별 수익률 감소 패턴 존재 여부

## 5. 보고서 11개 항목에 대한 트레이더 관점 보완

| 보고서 항목 | 트레이더 관점 보완 |
|------------|-------------------|
| 1. 종목별 타임라인 | 보유시간 구간별(~10분, ~30분, ~1시간, 1시간+) 성과 분류 추가 |
| 2. 놓친 기회 | 고점 캡처율뿐 아니라 "진입 후 최저점까지 MDD"도 표시 → 심리적 압박 측정 |
| 3. 시간대별 수익 | 위 시간대 6구간으로 세분화. 점심시간 매매 경고 플래그 |
| 5. 승률/손익비 | Profit Factor 외에 "기대값(Expectancy)" 추가: (승률×평균이익) - (패률×평균손실) |
| 6. 변동성 대비 성과 | ATR 기반 캡처율 산출: 실현수익 / (당일고가-당일저가) |
| 9. 반복매매 | 3회 이상 동일종목 매매 시 "경고" 플래그. 회차별 체감 수익률 그래프 |
| 11. 호가 분석 | 매매 시점 체결강도(매수체결/매도체결 비율) 추가 분석 |
