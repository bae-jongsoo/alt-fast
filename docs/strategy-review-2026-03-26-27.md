### 3가지 핵심 문제 (전원 일치)

**문제 1: 과다매매 (Overtrading)**
- 일 30~39건, 시간당 5~6건
- KB금융 하루 10회 반복매매, 수수료만 18,292원
- 매매 건수와 손실 간 양의 상관관계
- 3/26→3/27 매매 빈도 26% 증가, 성과는 62% 악화 → 틸트(tilt) 패턴

**문제 2: 초단타 함정**
- 대부분 1~5분 보유, 1~2틱 손실로 청산
- LLM 처리 >5초가 100% — 시그널 생성이 보유기간의 상당부분 차지
- 3/27 감시종목 +2.65% 상승장에서 -13.19% 손실 → 종목은 맞았으나 실행이 틀림
- 캡처율 "우수" 등급은 전체의 ~20%에 불과

**문제 3: 수급 신호 역선택**
- 수급 우위 시 승률 9.5% < 수급 열위 시 29.4%
- 모멘텀 추종 전략인데 실제 대형주 일중 가격은 평균회귀 성향
- 체계적으로 고점 매수 후 되돌림에 손절하는 구조

### Kelly Criterion 분석
- Kelly% = -0.89 (3/27 기준) → **어떤 포지션 사이징을 적용해도 장기적으로 파산하는 수학적 구조**

## 2. 전략 전환 방향 (합의사항)

### 핵심 전환: "시장보다 빠르게" → "시장보다 깊게 읽기"

**폐기:** 초단타/스캘핑 (1~5분 보유, 분봉 기반 진입)
**전환:** 이벤트 드리븐 데이트레이딩~스윙 (수시간~수일 보유)

### LLM의 Edge가 존재하는 영역 (전원 일치)

LLM의 강점은 "속도"가 아니라 **"비정형 정보의 해석 깊이"**에 있음.

**Edge가 있는 구체적 틈새:**
1. **해석 난이도가 높은 공시** — 전환사채 조건, 자회사 구조변경 등 숫자만 봐서는 판단 어려운 것. 시장이 30분~수시간 해석하는 동안 LLM은 2분 만에 판단 가능
2. **2차 연관 종목 추론** — "A사가 B사 인수" → A,B는 즉시 반응하나 C(B의 납품업체)는 10~30분 뒤 반응. LLM이 연결고리를 빠르게 추론
3. **뉴스 클러스터 임계점** — 개별 뉴스는 사소하나 3~5일간 같은 방향 뉴스 축적 시 방향성 판단
4. **복수 정보 종합** — 미장+환율+수급+뉴스를 동시에 읽고 종합하는 것은 룰 기반으로 어렵지만 LLM은 자연스럽게 가능

**Edge가 없는 영역 (진입 금지):**
- 단순 호재/악재 뉴스 반응 (1~5분 내 반영, HFT 영역)
- 틱/호가 단위 가격 예측
- 헤드라인만으로 방향이 명확한 뉴스

### 학술적 근거
- PEAD (Post-Earnings Announcement Drift): 실적 서프라이즈 방향으로 수일~수주간 추가 이동. 1960년대부터 발견, 여전히 유효 (Ball & Brown 1968)
- LLM 뉴스 감성분석이 전통 감성사전보다 유의미하게 높은 예측력 (Lopez-Lira & Tang 2023)
- 한국 자사주 매입 공시 후 20거래일 평균 초과수익 +3.8% (국내 연구)
- 단, 초과수익률은 연 2~5% 수준으로 크지 않음. 수수료/슬리피지 관리가 핵심

## 3. 새 전략 설계

### 전체 구조
```
[매일 장 전] 거시 레짐 판단
  → Risk-On / 중립 / Risk-Off 분류
  → Risk-Off면 매매 자제 (관망)

[장 중] 이벤트 트리거
  → 공시 발생 or 뉴스 클러스터 감지 or 분봉 이상 감지
  → 1차 퀀트 필터: 거래량 2배+, 가격 변동률 체크
  → 통과 시 LLM 분석 호출

[LLM 판단] — edge 집중 지점
  → 공시/뉴스 해석 + confidence score (1~10)
  → 8점 이상만 실행 (일 2~5건)

[포지션 사이징] confidence 기반 차등
  → 0.7~0.8: 기본 / 0.8~0.9: 1.5배 / 0.9+: 2배
  → Half-Kelly 적용, 동시 포지션 최대 3~5개

[청산] 규칙 기반 (LLM 개입 최소화)
  → 손절: -2% 기계적 (예외 없음)
  → 1차 익절: +3% 시 절반 매도
  → 나머지: trailing stop -1.5% 또는 3영업일 시간청산
  → 반대 이벤트 발생 시 LLM이 청산 판단
```

### 리스크 관리 프레임워크
- 일일 손실 한도: 총 자산의 -2% → 당일 매매 중단
- 연속 손실 서킷브레이커: 3연패 시 1시간 쿨다운, 5연패 시 당일 중단
- 종목별 일일 손실 한도: -1% 도달 시 해당 종목 당일 금지
- 동일 종목 재진입 쿨다운: 최소 30분

## 4. 정보 소스 및 데이터

### 아키텍처 — 수집과 전략 분리
```
[수집 프로젝트] (독립)
  미장 / 환율 / 수급 / 원자재 / 뉴스 / 공시 / 분봉 / 이상감지
  → DB or Redis에 적재
      │
      ├─ [전략 A] 기존 시스템 (프롬프트 개선, 확신도 필터)
      ├─ [전략 B] 이벤트 드리븐 스윙
      ├─ [전략 C] 뉴스 클러스터 기반
      └─ [전략 D] 거시 레짐 + 섹터 로테이션
```

### 정보 소스 (영향력순)

**Tier 1 — 반드시 활용 (직접적·즉각적)**
| 소스 | 한국 시장 영향 | 확보 방법 |
|------|---------------|----------|
| 미국 3대 지수 + 나스닥 선물 | 코스피 시가 갭의 60~70% 설명 | KIS API / 무료 API |
| 원/달러 환율 | 외인 수급 선행지표 | KIS API |
| 외국인/기관 수급 | 대형주 방향 결정 | KIS API |
| DART 공시 | 이벤트 드리븐 핵심 트리거 | 이미 수집 중 |
| 뉴스 (네이버) | 감성/클러스터 분석 | 이미 수집 중 |

**Tier 2 — 섹터별 판단**
| 소스 | 영향 |
|------|------|
| SOX (필라델피아 반도체지수) | 삼성전자, SK하이닉스 직결 |
| WTI 유가 | 정유/화학 수혜, 항공 타격 |

| DRAM/NAND 현물가 | 반도체 섹터 실적 기대 |
| 미국 10년물 금리 | 성장주/가치주 로테이션 |

**Tier 3 — 중기 레짐**
- FOMC, 한은 금통위, VIX, 고용지표, 경제지표 캘린더

### 분봉/호가 데이터 — 역할 전환

**판단(What)이 아니라 실행(When)으로 전환:**
| 역할 | 활용 |
|------|------|
| 진입 타이밍 최적화 | LLM "매수" 결정 후, VWAP/눌림목에서 진입 |
| 퀀트 필터 | 거래량 2배 이상? 호가 스프레드 정상? |
| 청산 보조 | 트레일링 스탑 모니터링, 모멘텀 소진 감지 |
| 이상 감지 센서 | 뉴스 없이 거래량 급증 → LLM에 원인 분석 요청 |

## 5. 뉴스/공시 → 가격 반영 타임라인 (한국 대형주)

```
[이벤트 발표]
  ├─ 0~30초: 알고리즘 반응 (전체의 40~50%)
  ├─ 30초~5분: 전업 트레이더 반응 (누적 60~70%)
  ├─ 5분~30분: 일반 투자자 유입 (누적 80~85%)
  ├─ 30분~수시간: 2차 해석, 연관 종목 파급 (누적 90~95%)
  └─ 1~3일: 완전 반영 (100%)
```

**LLM 진입 가능 구간:**
| 이벤트 유형 | 잔여 기회 | 판정 |
|------------|----------|------|
| 단순 호재/악재 | 없음 | 진입 금지 |
| 복합 공시 (M&A, 구조변경) | 수시간~수일 | 최적 영역 |
| 2차 연관 종목 | 10~30분 뒤 지연 반응 | 가능 |
| 뉴스 클러스터 | 수시간~수일 | 가장 안정적 |
| 실적 PEAD | 당일~수주 | 충분 (60~70%는 다음 날 이후) |
| 자사주/지분변동 | 수일~수주 | 매우 충분 |
